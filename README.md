# odin-recipes

Odin Recipes

Introduction

The Odin Recipes project is a beginner-friendly HTML exercise designed to strengthen foundational HTML skills. The goal is to create a basic recipe website consisting of a main index page that links to individual recipe pages. While the design will be minimalistic, this project focuses on building a solid understanding of HTML structure and best practices.

Features;

- A main index page with links to multiple recipe pages.
- Individual recipe pages showcasing:
    - A title.
    - An image of the dish.
    - A brief description.
    - A list of ingredients.
    - Step-by-step instructions.

Skills Demonstrated;

- Structuring HTML documents with semantic elements.
- Creating and organizing directories and files.
- Linking between pages using relative paths.
- Formatting content with headings, paragraphs, lists, and images.
- Using Git and GitHub to manage and share projects.
